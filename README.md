# PEAK British English Translation

This mod changes:

- Backpack -> Rucksack
- Luggage -> Suitcase

If you find any other American English terms in the game that you think should be changed to British English, please open an [issue](https://github.com/SavageCore/PEAK_BritishEnglish_Translation/issues) or a [pull request](https://github.com/SavageCore/PEAK_BritishEnglish_Translation/pulls).
